# 看影音

<p align="center">
  <img src="assets/images/logo/logo_rounded.png" alt="看影音图标" width="160">
</p>

看影音是一款面向 Windows 的本地与个人网盘视频媒体库。它可以扫描、整理和播放用户自己的视频文件，并通过 TMDB 补充中文标题、简介、评分、海报、背景图和季集信息。

项目专注用户自有媒体的整理、元数据管理与播放体验。

## 主要功能

### 媒体库

- 添加本地文件夹并递归扫描常见视频格式。
- 根据文件名和目录结构识别剧名、季度、集数、特别篇及独立电影。
- 将同一系列的视频整理为媒体卡片和选集列表。
- 本地海报墙会根据窗口宽度自动增减列数，海报保持 300 像素尺寸上限，最大化时不会跟随窗口放大。
- 支持本地封面、自定义封面、视频缩略图和媒体来源筛选。
- 删除媒体源、索引或缓存时不会删除用户的原始视频文件。

### OpenList、夸克与百度网盘

> OpenList 功能仍在调试，当前不建议使用。网盘功能仅用于访问用户自己的媒体，请保留重要数据的独立备份。

- OpenList 的连接、扫描和播放流程仍在调试，当前不建议使用。
- 可使用夸克网页版 Cookie 添加个人网盘数据源，扫描视频和外部字幕。
- 可填写自己的百度开放平台 API Key 和 Secret，通过官方 OAuth 授权添加多个百度账号，并为每个来源选择多个媒体目录。
- 百度来源严格只读，只读取目录、文件信息和原文件数据，不会重命名、移动、删除或上传百度网盘文件。
- 支持在线播放夸克媒体，以及手动查看、选择和转存夸克分享内容。
- 夸克默认转存目录会自动纳入媒体扫描范围；旧配置会在下次扫描时自动补齐，无需再次转存文件。
- 夸克与百度原文件通过公共本机分段中转边下边播，支持预读、拖动和临时鉴权自动更新；分段缓存会在退出或切集后清理。
- 左侧“网盘资源”可直接切换来源，并将全部媒体根目录中的合格视频汇总为海报墙，无需逐层打开文件夹。
- 打开网盘资源时会先显示上次媒体库索引，再在后台递归刷新；扫描期间已有海报和播放不受影响。
- 修改 OpenList 或夸克媒体根目录后，旧目录资源会实时从本地与网盘海报墙隐藏；看影音只扫描变化来源一次，完成后自动显示新目录资源。
- 网盘电影和剧集支持 TMDB 自动刮削，可显示海报、中文标题、评分和简介。
- 已刮削电影和剧集使用沉浸式海报卡；同一剧集的每一季独立显示对应 TMDB 季度海报，打开后只列出该季选集。
-当视频文件名仅包含纯数字集号时，可从作品和季度文件夹继承剧名、季度与集号；年份、4K、WEB-DL、H265、DV、HDR 和 Atmos 等发布规格不会作为剧名。
- “内嵌中字”“高码率”“第3季 - 2160p”等包装或规格目录会继承路径中的作品名；媒体根目录直接存放的纯集号文件及多季目录会归并为同一作品。
- 同一季度按 4K、1080P、内封字幕或内嵌字幕保存的多个版本会归入同一季度卡；卡片按唯一集号显示集数，选集仍保留并标注全部可播放版本。
- 同一集的不同网盘资源会统一显示相同的 `SxxExx`，不会再按列表位置误报成新集数；每个真实版本仍可分别选择播放。
- 同时配置作品目录和季度子目录时只建立一个作品身份；分集文件中的共同标题会优先用于 TMDB 搜索，待确认卡片可从状态标签或右上角菜单手动确认。
- 最大化窗口只会增加海报列数，海报保持尺寸上限；窄窗口自动减少列数，不产生横向滚动。
- 网盘选集显示应用内规范剧名和季集编号，真实原名与路径可在媒体详情中核对；播放仍使用原始远程 ID 和路径。
- 同一作品只请求一次 TMDB 详情，再向实际存在的季度分配海报；修改刮削名称会同步该作品的全部季度，不会影响其他目录或来源中的同名作品。
- 非视频文件和不大于“网盘媒体识别大小”的视频不会显示、刮削或参与自动继承，字幕仍会在播放时自动关联。
- 支持主动递归整理当前网盘来源，自动跳过已处理资源，并汇总 TMDB 批量匹配结果。
- 支持手动重新匹配与刮削当前来源；单项操作可先修改搜索词、类型和年份，再从完整候选信息中确认结果。
- 支持修改网盘作品的刮削名称；应用内规范名称不会重命名远程文件、移动目录或改变播放路径。
- 远程服务接口变化可能造成连接、扫描或播放兼容性问题，不建议将网盘作为唯一媒体来源。
- 删除网盘数据源时只会清理对应索引与缓存，不会删除远端原始文件。

OpenList、夸克和百度网盘在本项目中仅作为用户自有媒体的数据入口，不是在线影视搜索服务。

### TMDB 刮削

- 获取中文标题、原始标题、简介、评分、海报和背景图。
- 支持电影、电视剧、季度和剧集信息。
- 支持自动匹配、手动选择候选、重新匹配和单独刮削。
- 可配置语言、地区、匹配阈值以及标题和图片覆盖策略。
- 公共安装包不内置 TMDB Key；用户可在设置中填写自己的 Key，并由 Windows 安全存储保护。
- 没有可用 API Key、断网或 TMDB 不可用时，本地扫描和播放仍可使用。

### 播放器

- 基于 media-kit 与 mpv，支持常见本地和远程媒体格式。
- 支持 Windows 硬件解码、低内存模式和 TrueHD 兼容音轨回退。
- 支持播放进度、倍速、快进快退、画面比例、选集和自动连播。
- 支持全屏、画中画、控制面板锁定、截图和外部播放器。
- 支持定时停止和后台播放。
- TrueHD 音轨播放失败时可尝试切换到已有的兼容音轨。
- 提供 Anime4K 动画画质增强；效率档使用官方快速组合，质量档保留完整组合，仅在画面确实需要放大时自适应启用。

### 字幕与音轨

- 查看并切换内嵌音轨和字幕轨。
- 自动匹配同目录同名字幕，也可手动导入外部字幕。
- 支持网盘字幕下载和本地缓存。
- 可调整字幕字体、字号、颜色、描边、位置和时间偏移。
- 字幕延迟可按视频保存。

### 诊断

- 自动记录脱敏运行日志，最多保留 10 个日志文件。
- 支持导出脱敏诊断 ZIP，用于排查播放问题。

## 安装

1. 从当前仓库的 [Releases](https://github.com/ddddd-ren/KanYingYin/releases) 下载最新的 `看影音-版本号.msix`。
2. 双击 MSIX 并按 Windows 提示完成安装。
3. 如果系统提示签名证书不受信任，需要先信任发布页提供的项目证书，再重新安装。
4. 新安装或首次启动时，如果桌面或开始菜单中没有快捷方式，应用会询问是否创建。

看影音使用专属的应用身份和数据目录。

## 使用说明

1. 打开“本地”页面并添加包含视频的文件夹。
2. 等待扫描完成后，从媒体库选择影片或剧集播放。
3. 如需 TMDB 信息，可在“设置 > TMDB 刮削”中填写自己的 API Key；公共安装包不内置 TMDB Key。
4. 如需访问个人网盘，在“设置 > 网盘数据源”中添加 OpenList、夸克或百度网盘来源；百度使用自己的开放平台凭据和官方授权。
5. 播放器的解码、Anime4K、字幕和快捷键选项位于“设置 > 播放设置”与“操作设置”。

## 数据与隐私

- 应用不包含遥测或用户行为统计。
- 本地媒体索引、海报和字幕缓存保存在看影音自己的应用数据目录。
- 日志导出前会隐藏远程 URL 路径、请求头和常见凭据字段。
- 删除媒体源、媒体索引或缓存不会删除原始视频文件。
- OpenList 凭据、夸克 Cookie、百度开放平台凭据和 OAuth 令牌通过系统安全存储保存，不应写入日志或仓库。
- TMDB、OpenList、夸克与百度请求只在用户启用对应功能时发生。

## 开发与构建

项目使用 Flutter Modular 组织模块，使用 MobX 管理状态。项目约定的 Flutter SDK 为 `D:\flutter`，版本为 3.41.9。

### 恢复依赖

```powershell
D:\flutter\bin\flutter.bat pub get
```

### 测试与静态分析

```powershell
D:\flutter\bin\flutter.bat test --no-pub
D:\flutter\bin\flutter.bat analyze --no-pub
```

### Windows Release

```powershell
D:\flutter\bin\flutter.bat build windows --release --no-pub
```

### 生成 MSIX

MSIX 必须基于本轮生成的 Windows Release 目录封装。签名脚本从当前用户保护的凭据文件读取密码，不写入仓库，也不依赖或内置 TMDB Key。

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File tool\windows\build_signed_release.ps1
```

脚本会验证 `AppxManifest.xml` 中的版本、包身份和数字签名，并将 `看影音-版本号.msix` 与异机安装包复制到当前用户桌面。

## 项目结构

```text
lib/
  pages/          页面、播放器界面与控制器
  services/       本地扫描、TMDB、OpenList、字幕和缓存服务
  repositories/   媒体源、索引和元数据持久化
  modules/        媒体、剧集、播放请求等领域模型
  utils/          日志、存储、窗口和通用工具
windows/          Windows Runner、窗口行为和原生集成
test/             单元测试、组件测试和身份一致性测试
```
