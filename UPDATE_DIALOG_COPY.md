## 当前版本

- 安装包版本：1.0.2.0

## 弹窗标题

看影音 1.0.2 正式版

## 弹窗正文

- 修复新安装后只有开始菜单入口、首次启动却没有询问创建桌面快捷方式的问题。
- 首次启动发现桌面快捷方式缺失时会明确询问是否创建；选择“暂不创建”后会尊重选择，不会每次启动重复弹窗。
- 本地媒体库、夸克和百度网盘媒体库可继续扫描与播放个人媒体；TMDB 可补充中文资料，OpenList 功能仍在调试，当前不建议使用。
- 播放器的字幕、选集、全屏、硬件解码和 Anime4K 行为保持不变；没有 TMDB Key 或断网时，本地与网盘媒体库及播放器仍可使用。
- 异机安装包继续先验证签名并安全导入公钥；网盘凭据和 TMDB Key 使用 Windows 安全存储，应用数据保存在看影音专属目录，不会修改或删除原始媒体。
