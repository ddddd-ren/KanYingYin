## 当前版本

- 应用版本：1.0.0
- 安装包版本：1.0.0.0
- 日期：2026-07-23

## 弹窗标题

看影音 1.0.0 正式版

## 弹窗正文

- 看影音 Windows 1.0.0 正式版现已发布，专注整理和播放本地视频与个人网盘中的自有媒体。
- 本地媒体库支持递归扫描文件夹，识别电影、剧名、季度、集数和特别篇，并整理为海报墙与选集；还可使用本地封面、自定义封面和视频缩略图。
- 夸克与百度网盘可作为个人媒体来源，支持扫描视频与字幕、整理海报墙和边下边播；百度使用用户自己的开放平台凭据与官方授权。看影音不会修改网盘文件、目录或远程标识。
- OpenList 的连接、扫描和播放流程仍在调试，当前不建议使用。
- TMDB 可补充中文标题、简介、评分、海报、背景图和季集信息，支持自动匹配、手动确认与重新匹配；公共安装包不内置 TMDB Key。
- 播放器支持常见媒体格式、播放进度、倍速、快进快退、选集、自动连播、全屏、画中画、截图和外部播放器，并提供硬件解码、渲染器选择与 Anime4K 超分辨率。
- 支持切换内嵌字幕和音轨、自动匹配或手动导入外部字幕，以及调整字幕字体、颜色、位置和时间；未标注语言的轨道可在播放后按需确认。
- 应用不包含遥测；网盘凭据和 TMDB Key 使用 Windows 安全存储，诊断日志会隐藏远程地址、请求头和常见凭据，并支持在设置中导出。
- 没有 TMDB Key 或断网时，应用仍可启动，本地扫描和播放器不受影响，已缓存的本地与网盘媒体库可继续浏览。删除媒体源、索引或缓存不会修改或删除原始媒体。
- 首次启动时，如果桌面或开始菜单缺少快捷方式，应用会先询问是否创建；应用数据保存在看影音专属目录。
- 若电脑已安装版本号高于 1.0.0 的预览版，需要先卸载预览版再安装本正式版；卸载前请按需备份应用设置和索引数据。
