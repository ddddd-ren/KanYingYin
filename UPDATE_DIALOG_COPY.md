## 弹窗正文

- 本测试版继续修复夸克默认转存目录已保存、扫描却仍可能漏掉其中视频的问题。
- 仅保存了旧路径或旧远程 ID 的来源会在下次扫描时自动换成真实转存目录，无需重新选择或再次转存文件。
- 在转存目录自愈或写入扫描状态期间取消任务时，不再残留“正在扫描”；脱敏日志可帮助核对扫描结果。
- 没有 TMDB Key、断网或暂时连接不到 TMDB 时，本地与网盘媒体库、扫描和播放器仍可使用。
- 本次不会修改网盘文件，也不会删除用户的原始媒体；播放器的字幕、选集、全屏、硬件解码和 Anime4K 行为保持不变。
