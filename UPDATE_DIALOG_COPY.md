## 弹窗标题

看影音 2.1.31 测试版

## 弹窗正文

- 本测试版将用户填写的 TMDB Key 迁移到 Windows 安全存储，并只向 TMDB 官方服务发送；没有 Key 或断网时，本地媒体库扫描和播放器仍可使用。
- 日志查看、复制、清理和控制台输出现在统一脱敏；缓存统计与清理在目录不存在或删除失败时也能给出可靠结果。
- Windows 外部播放器现在支持中文、日文等本地路径并自动清理临时播放列表；启动时会根据桌面和开始菜单入口状态决定是否询问创建快捷方式。
- 本次不会删除本地视频、字幕、封面或网盘文件；应用启动、媒体库和播放器原有功能保持可用。
