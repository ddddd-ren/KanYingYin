# 看影音 1.0.1 累积正式版实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将当前 `2.1.55` 代码统一为 `1.0.1` 正式版，生成并验证 Windows 安装包，推送 `main`，创建 `v1.0.1` GitHub Release 并更新下载附件与描述。

**Architecture:** 本次不改变媒体库、播放器或网盘业务实现，只调整版本元数据、正式版用户文案和相关测试。发布门禁按“测试与静态分析 → Windows Release → 签名 MSIX 与异机安装包 → Git 提交与推送 → GitHub 草稿 Release 附件校验 → 正式发布”顺序执行；任何前置校验失败都停止后续发布动作。

**Tech Stack:** Flutter 3.41.9、Dart、Flutter Test、PowerShell、MSIX、Git、GitHub CLI/REST API。
