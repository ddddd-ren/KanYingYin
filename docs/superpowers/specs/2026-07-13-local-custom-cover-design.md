# 本地视频自定义封面设计

## 目标

让用户能在本地视频剧集的操作菜单中选择一张本地图片，作为该剧集的自定义封面。

## 方案

在 `LocalPage` 的剧集操作底部菜单中新增“自定义封面”。用户选择 JPG、JPEG、PNG 或 WEBP 图片后，应用将图片复制到首集所在目录，并命名为 `cover` 加原扩展名，例如 `cover.jpg`。

`LocalCoverFinder` 已将 `cover` 设为常用封面名，且优先于缩略图，因此无须新增封面存储模型。复制完成后刷新本地目录，列表立即展示新封面。用户取消选择时不改变原有封面；复制失败时给出失败提示。

## 范围

- 仅增加剧集级自定义封面，不增加单集级封面选择。
- 复用已有 `file_picker` 依赖和本地封面识别规则。
- 增加可独立测试的文件复制服务，验证目标命名与替换已有封面。
- 升级应用、MSIX 和接口版本至 2.3.60，并同步用户更新日志。
