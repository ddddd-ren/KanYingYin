# TMDB 刮削选项与桌面界面调整设计

## 目标

为看影音增加可持久化的 TMDB 刮削选项，并允许用户在单次手动刮削时临时覆盖部分全局设置。自动扫描继续使用全局默认值，现有字段锁定规则保持最高优先级。同时补齐桌面窗口控制栏，并删除关于页面中不属于独立本地播放器的外部链接。

## 用户入口

### TMDB 设置页

在现有 API Key、自动刮削、连接测试和缓存清理之外增加“刮削选项”区域：

- 首选语言：简体中文、繁体中文、英语，默认简体中文。
- 媒体类型：自动判断、电影、电视剧，默认自动判断。
- 自动匹配置信度：保守、标准、宽松，默认标准。
- 覆盖已有标题：默认关闭。
- 覆盖已有简介：默认开启。
- 覆盖已有海报：默认开启。
- 获取海报：默认开启。
- 获取背景图：默认开启。

所有选项与 API Key 一并由“保存”按钮写入本机设置。

### 单次手动刮削

用户从媒体系列菜单选择“刮削信息”或“重新匹配”时，先显示简洁的刮削选项弹层：

- 媒体类型可以临时选择自动、电影或电视剧。
- 可以临时控制是否覆盖标题、简介和海报。
- 默认值来自全局设置。
- 临时选择只影响本次操作，不回写全局设置。

自动扫描不显示弹层，直接使用全局设置。

### 文件组操作菜单

本地文件浏览页的视频操作底部菜单增加“TMDB 刮削”，位于“在线查找封面”之前。该入口执行完整元数据刮削：

- 根据当前文件组中的视频路径定位媒体库索引中的真实系列名称。
- 打开与媒体库系列菜单相同的单次刮削选项弹层。
- 高置信度结果直接保存标题、简介、评分、海报和背景图。
- 低置信度结果继续打开候选确认弹层。
- 用户确认候选后更新同系列全部索引条目。
- 当前文件组尚未建立媒体库索引时，不发起无效请求，提示用户先扫描媒体库。

## 配置模型

新增强类型 `TmdbScrapeOptions`，负责默认值、设置读取和序列化。字段包括：

- `language`
- `mediaTypeMode`
- `confidenceMode`
- `overwriteTitle`
- `overwriteOverview`
- `overwritePoster`
- `fetchPoster`
- `fetchBackdrop`

置信度映射固定为：

- 保守：最低分 0.90，领先第二候选至少 0.15。
- 标准：最低分 0.80，领先第二候选至少 0.10。
- 宽松：最低分 0.70，领先第二候选至少 0.05。

## 数据流

1. `LocalController` 从本机设置读取全局 `TmdbScrapeOptions`。
2. 自动扫描把全局选项传给 `LocalTmdbScrapeService`。
3. 手动刮削弹层生成临时选项，再传给同一服务。
4. 服务根据媒体类型选项决定自动推断、电影或电视剧。
5. `TmdbScraper` 使用首选语言和置信度阈值搜索、匹配及读取详情。
6. 服务根据图片获取和覆盖选项合并元数据，并更新同系列全部条目。

## 覆盖规则

优先级从高到低：

1. 字段锁定：`titleLocked`、`overviewLocked`、`posterLocked` 永远禁止覆盖。
2. 获取开关：关闭海报或背景图获取时，不写入对应新值。
3. 覆盖开关：已有值存在且对应覆盖开关关闭时保留旧值。
4. 其他未锁定且允许覆盖的字段使用本次 TMDB 结果。

手动选择低置信度候选后同样遵守上述规则。

## 错误处理

- API Key 为空时跳过刮削，不影响扫描和播放。
- 网络或 TMDB 请求失败时保留已有元数据，并标记刮削失败。
- 没有高置信度结果时进入候选确认，不自动写入错误结果。
- 图片字段缺失时保留已有图片，不写入空值。
- 日志不得记录 API Key、v4 令牌或带认证参数的完整 URL。

## 桌面窗口控制栏

桌面端无边框窗口右上角固定显示以下控制按钮：

- 窗口置顶：切换窗口始终置顶状态，并同步更新图标和提示。
- 最小化：最小化当前窗口。
- 最大化/还原：根据当前窗口状态显示对应操作。
- 关闭：关闭应用窗口，悬停时使用明确的红色危险状态。

控制栏仅在 Windows、macOS 和 Linux 显示。按钮尺寸固定，不挤压页面标题或返回按钮。现有窗口拖动区域、播放器全屏和退出全屏行为保持不变，默认不启用窗口置顶。

## 关于页面清理

删除关于页面中的整个“外部链接”区域，包括项目主页、代码仓库、图标创作、番剧索引和以图搜番。保留应用名称、版本、更新日志、许可证及其他属于看影音自身的信息。

## 测试

- 配置默认值和设置序列化往返测试。
- 三档置信度阈值测试。
- 强制电影、强制电视剧和自动推断测试。
- 首选语言传递及英语回退测试。
- 标题、简介、海报分别覆盖与保留测试。
- 锁定字段优先于覆盖设置测试。
- 关闭海报或背景图获取时不写入对应字段测试。
- 自动扫描使用全局选项，手动刮削使用临时选项测试。
- 桌面窗口控制按钮的显示条件、置顶状态和最大化状态测试。
- 关于页面不再包含外部链接区域的回归测试。
- 全量测试、静态检查、Windows Release 和 MSIX 验证。

## 版本与交付

该功能进入用户安装包时同步更新应用版本、MSIX 版本、发布说明和应用内版本更新日志。最终 MSIX 复制到桌面，并验证签名与清单版本。
