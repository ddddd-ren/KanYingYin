# 分离季度与特别篇收纳设计

## 目标

保留本地页面的剧集收纳能力，但不再跨季度、OVA、特别篇和剧场版合并。每个季度或作品类型独立显示，同一季度内部的剧集继续收纳。

## 用户界面

- 同一季度的普通剧集合并为一张卡片。
- 第一季、第二季等不同季度分别显示。
- OVA、OAD、特别篇和剧场版分别显示，不与季度正片合并。
- 恢复“扫描媒体库”和“媒体库”按钮及系列数量。
- 保留选择目录、媒体源、最近目录、刷新、海报、媒体信息、缩略图和 TMDB 刮削入口。

## 播放行为

- 点击卡片时创建当前季度或当前作品类型的播放会话。
- 选集只包含当前季度、OVA、特别篇或剧场版中的视频。
- 字幕自动匹配、硬件解码、Anime4K、播放历史和播放器动画保持不变。

## 数据处理

- 不删除已有媒体索引、TMDB 元数据、海报、缩略图或用户视频。
- 调整 `LocalSeriesGrouper` 的分组键，使基础作品名相同但季度或类型不同的内容分开。
- 媒体库构建器继续复用同一分组规则。

## 验证

- 回归测试确认同一季度的多集仍合并。
- 回归测试确认不同季度、OVA 和剧场版分别显示。
- 回归测试确认播放列表只包含当前分组的视频。
- 完整执行 `flutter test`、`flutter analyze` 和 Windows Release 构建。
- 实机检查本地页面的视频数量和点击播放行为。

## 版本交付

- 同步更新发布说明和应用内版本历史。
- 生成并验证桌面安装包 `看影音-1.2.5.msix`。
