# 字幕时间偏移设计

## 目标

在现有播放器字幕设置面板中增加字幕时间偏移调节。用户可以让字幕提前或延后出现，调整立即生效，并按视频独立保存。

## 交互设计

- 在“字幕设置”面板的字幕样式设置之前增加“字幕时间”区域。
- 使用范围为 `-30.0` 到 `+30.0` 秒的滑块，步进为 `0.5` 秒。
- 提供“提前 0.5 秒”和“延后 0.5 秒”两个图标按钮。
- 显示当前偏移值及含义：负值为字幕提前，正值为字幕延后，零值为同步。
- 提供重置按钮，将当前视频的字幕偏移恢复为 `0.0` 秒。
- 控件沿用现有字幕设置面板的布局、颜色和动画方式。

## 状态与存储

- `PlayerController` 持有当前字幕偏移，单位为秒，范围限制在 `-30.0` 到 `+30.0`。
- 调节时立即把偏移同步给 `media_kit`/MPV 字幕延迟属性。
- 使用规范化视频路径作为键，保存每个视频自己的字幕偏移。
- 打开视频或切换剧集时读取该视频的偏移；没有记录时使用 `0.0` 秒。
- 重置后删除该视频的持久化记录，避免保存无意义的零值。
- 字幕开关、选择字幕和样式重置不改变当前视频的时间偏移。

## 数据流

1. 播放器载入视频并取得规范化视频路径。
2. 控制器从本地设置读取该路径对应的字幕偏移。
3. 控制器将偏移同步到播放器字幕延迟属性。
4. 用户操作滑块或按钮时，控制器限制范围、更新可观察状态、同步播放器并保存。
5. UI 观察控制器状态并立即更新数值和按钮可用状态。

## 异常处理

- 播放器尚未初始化时仍保存偏移，播放器可用后再次同步。
- 原生字幕延迟属性设置失败时记录日志，不中断播放或关闭设置面板。
- 存储数据缺失、格式错误或超出范围时回退到 `0.0` 秒并执行范围限制。

## 测试

- 验证偏移按 `0.5` 秒调整并限制在 `-30.0` 到 `+30.0` 秒。
- 验证不同视频保存和恢复各自的偏移。
- 验证无记录视频默认为 `0.0` 秒。
- 验证重置删除当前视频记录并立即同步零值。
- 验证字幕设置界面包含滑块、提前、延后和重置操作。
