# 媒体库整理与清理 Skill 设计

## 目标

创建全局 Skill `media-library-curator`，用于扫描用户明确指定的媒体目录，依据 TMDB 简体中文标题提出命名与归并方案，并将非正片文件分为可直接清理、确认后清理和禁止清理三类。

Skill 不设置默认媒体目录。用户未提供路径时必须先询问路径，不得猜测或扫描其他目录。

## 安装结构

安装到 `C:\Users\asus\.codex\skills\media-library-curator`：

```text
media-library-curator/
├── SKILL.md
├── agents/
│   └── openai.yaml
└── scripts/
    └── scan_media.ps1
```

- `SKILL.md`：定义工作流、TMDB 命名规则、版本归并规则和清理安全边界。
- `scan_media.ps1`：只读扫描指定目录，输出媒体统计和分类候选，不执行删除或移动。
- `agents/openai.yaml`：提供技能显示名称、简介和默认提示。

## 工作流

1. 要求用户提供明确的媒体库路径。
2. 验证路径存在且为目录；在 PowerShell 中启用 UTF-8 输出。
3. 运行只读扫描脚本，统计视频、字幕、图片、压缩包、元数据和其他文件。
4. 识别电影、电视剧季集、字幕对应关系和同一影片的不同版本。
5. 需要改名时，先核对 TMDB `zh-CN` 标题，再展示完整改名计划和冲突。
6. 需要清理时，输出分类、理由、绝对路径和大小；默认不删除。
7. 仅在用户明确授权具体候选后执行删除，并在操作后复核文件数量和剩余内容。

## 命名规则

- 电影文件夹和主视频使用 TMDB 简体中文标题。
- 电视剧使用 TMDB 剧集标题，并保留季号和集号。
- 字幕主文件名与对应视频一致，保留语言标识和字幕扩展名。
- 同一影片的普通版和导演剪辑版共用一个作品文件夹，以文件名后缀区分版本。
- 由正剧单集或连续多集重新剪辑而成的导演剪辑版、加长版和重剪版，必须放入对应电视剧的正剧文件夹；存在季目录时放入对应季目录，不得建立独立顶层作品文件夹。
- 正剧衍生版本的文件名保留 TMDB 剧集标题、集数或集数范围及版本标识，例如 `假面骑士OOO 第47-48集（导演剪辑版）.mkv`。
- TMDB 无独立条目的特别篇、最终章或剪辑版不得强行匹配其他条目。
- 改名或移动前必须检查目标冲突，禁止覆盖已有文件。

## 清理分类

### 可直接清理候选

- 文件名明确包含下载站网址、推广语或“更多下载”“请访问官网”等广告特征的文件。
- 明确为网站推广内容的小体积广告视频。
- `RARBG.txt` 等发布组说明文件。
- 下载站广告文档、快捷方式和推广图片。

即使属于此类，Skill 也只列为候选，不自动删除。

### 确认后清理候选

- 已确认完整解压且字幕文件齐全的 `.zip`、`.rar`、`.7z` 字幕包。
- 重复或过期的发布组 `.nfo`。
- 不再需要的音轨说明文本。
- 正片完整时的 `Sample` 样片。
- 已保留有效海报后的重复海报。

### 禁止清理

- 正片、电视剧集和有效的不同剪辑版本。
- 位于正剧文件夹内、由对应集数衍生的导演剪辑版、加长版和重剪版。
- `.ass`、`.srt`、`.ssa`、`.vtt`、`.sub`、`.sup` 字幕。
- TMDB 无条目但属于作品内容的特别篇、最终章和剪辑版本。
- 无法仅凭证据确认用途的文件。
- 仅因体积小、名称异常或 TMDB 无结果而被怀疑的文件。

## 安全要求

- 扫描脚本必须只读，不包含删除、移动或改名参数。
- 所有路径使用 `-LiteralPath`，避免方括号和通配符误匹配。
- 所有文件读写使用 UTF-8；PowerShell 输出先切换到 UTF-8。
- 删除操作不由脚本提供，必须由代理在用户明确授权后针对绝对路径执行。
- 删除前后分别统计文件数量；冲突、缺失或分类不确定时停止操作。

## 验证

采用 RED-GREEN-REFACTOR 验证：

1. 无 Skill 基线测试：验证代理是否会误删小体积正片、TMDB 无条目内容或未解压字幕包。
2. 脚本测试：覆盖中文路径、空目录、不存在目录、广告视频、字幕包、版本文件和样片。
3. 归并测试：电影剪辑版归入电影文件夹，正剧集数衍生剪辑版归入对应剧集文件夹或季目录。
4. 有 Skill 测试：相同场景下应正确分类，且任何清理请求都先输出绝对路径并等待授权。
5. 运行 Skill 初始化工具、元数据生成工具和 `quick_validate.py`，确认目录结构和 frontmatter 有效。

## 验收标准

- 未提供目录时不扫描。
- 扫描脚本不修改任何文件。
- 正片、字幕和有效版本不会进入直接清理候选。
- 正剧集数衍生版本不会作为独立顶层作品保留，也不会进入清理候选。
- 可直接清理与确认后清理的理由清晰且路径完整。
- TMDB 命名计划不会覆盖文件，无法唯一匹配时保持原名。
- Skill 可被 Codex 正确发现并通过官方校验工具。
