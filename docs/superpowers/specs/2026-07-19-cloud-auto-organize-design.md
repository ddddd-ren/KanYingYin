# 网盘资源自动批量整理设计

## 背景

看影音目前会在用户进入网盘目录后，对当前目录中可识别的资源执行后台 TMDB 尝试，也提供“刮削当前目录”按钮。该机制只覆盖当前已打开目录，不能让用户主动整理整个来源，也没有递归发现、确认提示和来源级结果汇总。

本功能新增用户主动触发的“自动批量整理”，递归检查当前网盘来源的配置根目录，使用 TMDB 简体中文元数据更新看影音中的显示名称。任何操作都不得重命名、移动或删除夸克/OpenList 中的文件和目录。

## 方案比较

### 方案 A：用户主动触发来源级递归整理（采用）

工具栏新增入口，确认后递归扫描当前来源并自动处理高置信度匹配。用户能控制 TMDB 请求时机，适合大型网盘，并可明确展示扫描和整理进度。

### 方案 B：每次进入来源时自动递归整理

操作最少，但每次打开页面都可能产生大量网盘和 TMDB 请求，容易触发限流，也难以让用户判断程序正在做什么。

### 方案 C：只增强“刮削当前目录”

改动最小，但用户仍需逐层进入目录并重复点击，不能达到“批量整理整个来源”的目标。

## 用户流程

1. 用户选择网盘来源。
2. 点击工具栏“自动批量整理”。
3. 弹窗说明整理范围、TMDB 请求和安全边界；用户确认后开始。
4. 页面先显示“正在扫描目录”，再显示“正在整理 x/y”。
5. 完成后显示成功、待确认、无结果、失败和已跳过数量。
6. 当前页面中受影响的卡片原位更新中文标题、海报和状态；当前路径、搜索词与播放行为不变。

按钮在目录加载、当前目录后台刮削、来源级整理期间禁用，防止同一资源并发重复匹配。

## 递归候选发现

新增职责单一的 `CloudResourceAutoOrganizer` 扫描服务。它只读取 `CloudDriveClient.listDirectory`，不调用播放解析、下载、重命名、移动或删除接口。

从当前来源的所有配置根目录开始，使用队列顺序遍历：

- 配置根目录中的独立视频作为独立影片候选。
- 非根目录如果直接包含视频，则该目录作为一个电影或剧集候选，不再把目录内的每集视频分别提交 TMDB。
- 非根目录如果包含“第一季”“Season 1”“S01”等季目录，则父目录作为剧集候选，不再向下把每季当成独立作品。
- 只包含普通子目录且没有直接视频的目录视为分类目录，继续向下遍历。
- 字幕、图片、文档和无法识别的文件不会成为 TMDB 候选。
- 候选通过来源 ID、远程 ID 和规范化路径去重。

为避免配置错误导致无限或超大遍历，单次最多读取 1000 个目录，最大深度 20。超过限制时停止并显示明确提示，不提交不完整结果为成功。

## TMDB 整理规则

- 已匹配且远程显示名称未变化的记录直接跳过。
- 七天内已确认无结果且名称未变化的记录跳过，沿用现有重试间隔。
- 正在被当前目录后台任务处理的资源不启动来源级整理；用户稍后可重试。
- 其余候选使用当前 TMDB 语言、类型和置信度设置调用既有 `scrape` 流程。
- 高置信度自动匹配写入中文标题、简介、评分、年份、海报和媒体索引。
- 存在候选但不能唯一确认时计为“待确认”，保留原显示名，用户仍可从卡片菜单手动匹配。
- 没有候选时计为“无结果”；单项错误计为“失败”，不会中断其他候选。
- 没有 TMDB API Key 时在扫描前提示用户配置，不读取网盘目录。

显示标题优先级保持：自定义标题、TMDB 中文标题、真实网盘名称。真实名称继续作为副标题，因此“整理”不会让用户失去远程文件身份信息。

## 类型与状态

新增强类型对象：

- `CloudResourceAutoOrganizePhase`：`scanning`、`scraping`。
- `CloudResourceAutoOrganizeProgress`：阶段、已扫描目录、已发现候选、当前项和总项。
- `CloudResourceAutoOrganizeSummary`：成功、待确认、无结果、失败、跳过数量。

`CloudResourcesController` 负责创建并关闭网盘客户端、调用扫描服务、过滤缓存记录、顺序提交 TMDB，并通过进度回调向页面报告。页面只显示状态和结果，不直接遍历网盘或拼装领域规则。

## 错误与取消边界

- 单个目录读取失败计入失败目录并继续扫描其他分支；根目录读取失败也不影响其他配置根目录。
- 如果所有根目录都无法读取，返回失败而不是“0 项成功”。
- 页面销毁、来源切换或控制器释放后不再更新旧页面状态。
- 整理失败不写入目录加载错误，不妨碍当前目录浏览和视频播放。
- Cookie、TMDB API Key、播放直链和请求头不会进入进度对象、日志、结果或持久化记录。

## 测试

- 扫描服务识别根目录独立视频、电影文件夹、含季目录的剧集和分类目录。
- 扫描服务不把单集、字幕、季目录或分类目录误当独立作品。
- 多根目录、重复路径、目录读取失败、深度和数量限制。
- 控制器跳过已匹配与近期无结果记录。
- 控制器汇总成功、待确认、无结果、失败和跳过，并继续处理单项失败。
- 无 API Key 时不读取网盘。
- 页面确认弹窗、安全文案、扫描/整理进度、禁用重复点击和完成汇总。
- 整理后卡片显示 TMDB 中文标题且保留真实网盘名称。
- 夸克播放、目录导航、当前目录刮削和本地媒体库回归。
