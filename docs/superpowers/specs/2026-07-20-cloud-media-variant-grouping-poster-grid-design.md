# 网盘剧集版本归并与固定海报尺寸设计

## 背景

当前夸克索引同时配置了作品目录及其季度子目录，导致同一季度被建立多个作品身份。《回魂计》目录下还按 `4K 高码率`、`1080P 内封简繁英`、`1080P 内嵌中字` 保存了同一批剧集，现有识别将这些版本目录当成作品名，继而把 `The Resurrected` 错配为 1991 年电影《死而复生》。

宽屏海报墙使用固定列数，窗口最大化后会把每张卡片拉宽，降低同屏信息密度。

## 目标

- 将 `The Resurrected` 正确识别并匹配为《回魂计》。
- 同一作品、季度和集号只建立一个逻辑剧集，同时保留全部真实网盘版本。
- 《回魂计》第 1 季只显示一张卡片，集数按唯一集号统计为 9 集。
- 重叠配置的父子媒体根不再产生重复作品和季度卡片。
- 海报保持当前普通窗口下的尺寸上限，宽屏仅增加列数。
- 为待确认资源提供清晰可见的手动确认入口。
- 不修改网盘原始文件、目录、远程 ID 或播放路径。

## 非目标

- 本轮不修改夸克播放接口、链接刷新或网络请求头。
- 不删除或隐藏任何有效视频版本。
- 不为《回魂计》编写作品专用硬编码分支。

## 识别与作品边界

扫描器先规范配置根目录：如果一个已配置根完整包含另一个已配置根，媒体树只使用最上层根建立作品身份，子根仍参与目录扫描，但不再独立创建作品。

版本目录识别扩展为结构化规则，覆盖以下组合：

- `4K 高码率`、`1080P` 等清晰度或码率描述。
- `内封简繁英`、`内嵌中字` 等字幕描述。
- `全9集` 等集数描述。
- 上述内容使用方括号、书名号、空格或连接符组合的目录名。

这些目录只提供版本标签，不参与作品标题候选。

当视频文件名包含稳定的剧名和季集号时，例如 `The.Resurrected.S01E01`，扫描器从同组文件提取共同标题 `The Resurrected`，并将其作为 TMDB 的优先搜索候选。目录标题保留为辅助候选，无法提取共同标题时不强制匹配。

## 剧集版本归并

逻辑剧集键由作品身份、季度和集号组成。同一逻辑剧集可以关联多个真实视频版本，每个版本继续保留：

- 原始远程 ID 与路径。
- 原始文件名与应用内虚拟名称。
- 分辨率、来源、编码、动态范围、音频和字幕标签。

季度卡片的集数使用唯一逻辑集号数量，不累计版本数量。打开选集后仍显示全部可播放版本，同集版本相邻排序，并在标题中显示版本标签，例如：

- `S01E01 · 4K 高码率`
- `S01E01 · 1080P 内封字幕`
- `S01E01 · 1080P 内嵌中字`

没有可区分标签时回退显示原文件名，避免不同版本看起来完全相同。

## 海报墙布局

海报网格使用最大横轴尺寸委托，不再使用宽屏固定四列。卡片最大宽度以当前普通窗口尺寸为基准，约为 300 像素，保持现有宽高比、间距、圆角、遮罩和交互。

- 窗口变宽：自动增加列数，卡片宽度不超过上限。
- 窗口变窄：自动减少列数，允许卡片在必要时缩小。
- 页面保持纵向滚动，不引入横向滚动。

## 手动确认入口

状态为“需要确认”的媒体卡提供两个等价入口：

- 卡片右上角菜单显示“手动确认匹配”。
- “需要确认”状态标签可以直接点击。

两个入口都打开现有 TMDB 候选确认窗口，并预填当前识别标题、媒体类型、年份和季集信息。确认后立即更新同一作品的季度卡片、海报和标题；取消操作不修改任何索引或网盘数据。普通“TMDB 刮削”和“重新匹配”入口继续保留。

## 数据流

1. 网盘客户端递归读取所有配置根。
2. 索引器保存完整目录快照，但向媒体树传入规范后的作品根集合。
3. 媒体树忽略版本目录标题，从视频文件名提取作品、季度、集号与版本标签。
4. 索引条目继续一对一对应真实视频文件。
5. 资源集合层按逻辑剧集键计算唯一集数并排列多个版本。
6. TMDB 使用共同文件标题优先匹配，成功后为作品各季度提供对应海报。
7. 待确认卡片通过菜单或状态标签进入同一候选确认流程。
8. 海报墙按最大卡片宽度自适应列数。

## 失败与回退

- 文件标题不一致或无法唯一提取时，保留目录标题并进入待确认状态。
- 同集版本标签缺失时保留原文件名，不丢弃视频。
- TMDB 无结果、无 API Key 或断网时，媒体树、选集和播放仍可使用。
- 目录扫描部分失败时沿用已有缓存，不清空成功索引。
- 版本归并只影响展示和逻辑计数，不修改远程数据。

## 测试与验收

自动化测试覆盖：

- 父作品根与季度子根同时配置时只建立一个作品身份。
- `4K 高码率` 和多组方括号版本目录不进入剧名。
- `The Resurrected` 成为优先 TMDB 搜索候选。
- 三个清晰度版本归入《回魂计》第 1 季的 9 个唯一集号。
- 选集保留 27 个真实文件并按集号、版本排序。
- 《弥留之国的爱丽丝》不再因重叠根产生重复季度卡片。
- 独立电影和不同作品不被跨目录合并。
- 待确认卡片的菜单和状态标签都能打开候选确认窗口，取消时不写入索引。
- 宽屏卡片宽度不超过上限且列数增加，窄屏列数减少且无横向滚动。
