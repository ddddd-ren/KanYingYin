# 字幕语种识别扩展设计

## 目标

让播放器的内嵌字幕菜单正确展示常见字幕语种与字幕特性，减少“未知语种”条目，同时不改变现有中文字幕自动选择策略。

## 背景

`EmbeddedTrackInfo` 当前仅识别中文变体、粤语、台配与英语。许多媒体容器使用标准语言代码或英文轨道标题，例如 `ja`、`jpn`、`es-419`、`Latin America`，现有逻辑会将它们展示为“未知语种”。截图中 `SDH` 也被附加在未知语种标题后，难以区分语种与无障碍字幕标记。

部分轨道的语言和标题均为空，或语言为 `und`。本迭代不读取字幕正文，因此这类轨道继续明确显示“未知语种 · 轨道 N”，避免猜测造成错误选择。

## 设计决定

### 1. 本地规则识别

在 `lib/pages/player/models/embedded_track_info.dart` 中扩展 `TrackLanguageKind`，以强类型枚举表示新增语种。识别顺序为：

1. 先解析明确的语言代码与区域变体；
2. 再匹配常见的中英文轨道标题别名；
3. 无匹配时返回 `unknown`。

识别输入继续由 `language` 与 `title` 组成。比较前统一转为小写，并将连字符、下划线和斜杠视为语言代码边界，防止 `en` 等短代码发生子串误匹配。

### 2. 覆盖范围

新增识别以下常见语种及变体：

- 日语：`ja`、`jpn`、`japanese`、`日语`；
- 韩语：`ko`、`kor`、`korean`、`韩语`；
- 西班牙语：`es`、`spa`、`spanish`、`西班牙语`；
- 拉丁美洲西班牙语：`es-419`、`es-mx`、`latin america`、`latam`、`latin american spanish`；
- 葡萄牙语及巴西变体：`pt`、`por`、`pt-br`、`brazilian portuguese`；
- 法语、德语、意大利语、俄语、乌克兰语、泰语、越南语、印尼语、马来语、阿拉伯语、印地语、土耳其语、荷兰语、波兰语、瑞典语、丹麦语、挪威语、芬兰语、希腊语、希伯来语、捷克语、罗马尼亚语、匈牙利语。

中文变体规则同步完善：`zh-TW`、`zh-HK`、`zh-MO`、`zh-Hant` 归为繁体中文；`zh-CN`、`zh-SG`、`zh-Hans` 归为简体中文。已有的简繁双语、简体中文与繁体中文标题别名保持有效。

### 3. 展示规则

语种主标题采用中文优先、英文辅助的方式，例如：

- `西班牙语（拉丁美洲） / Spanish (Latin America)`；
- `日语 / Japanese`；
- `葡萄牙语（巴西） / Portuguese (Brazil)`。

`SDH`、`CC`、`Forced`、`Signs & Songs` 等为字幕特性，不参与语种识别。它们会以简洁中文或原始缩写显示在 `detailLabel`，与 `SRT`、`ASS`、`PGS` 等编码信息并列。

语言和标题均无法识别时，主标题继续显示“未知语种 · 轨道 N”。若有非语种标题，则保留该标题，便于用户手动判断轨道用途。

### 4. 默认选择

`selectPreferredSubtitleTrack` 的优先级保持为：简体中文 → 简繁双语 → 繁体中文 → 媒体默认轨道。新增语种只改善菜单展示，不会自动覆盖默认轨道或改变无中文字幕时的行为。

## 错误处理与兼容性

- 无效、空白、`und`、`zxx`、`mul` 语言标签不作推断，保持未知；
- 识别逻辑为纯内存计算，不访问网络、不读取原始视频或字幕文件；
- 不修改播放器切换调用、动画、字幕渲染或硬件解码逻辑。

## 测试与验收

在 `test/embedded_track_info_test.dart` 新增测试，覆盖：

- 常见 ISO 代码、BCP-47 区域代码和中英文标题别名；
- `Latin America` 显示为拉丁美洲西班牙语；
- `SDH`、`CC`、`Forced` 显示在详情而不替代语种；
- 空语言、空标题和 `und` 仍显示未知语种；
- 现有中文字幕优先级与媒体默认轨回退策略不变。

验收命令为 `flutter test test/embedded_track_info_test.dart`、`flutter test`、`flutter analyze`、Windows Release 构建。作为可交付版本时，还需按项目规范更新版本、更新日志、生成并验证 MSIX、复制安装包到桌面。
