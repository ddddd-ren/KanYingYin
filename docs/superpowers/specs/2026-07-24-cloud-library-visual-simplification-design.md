# 网盘媒体库视觉精简设计

## 背景

网盘媒体海报卡右上角的圆形三点操作按钮面积偏大，容易抢占海报视觉焦点。网盘媒体库内容区顶部的目录导航条也不再需要，会增加界面层级和视觉负担。

## 目标

- 保留海报卡原有三点按钮、圆形外观、菜单内容和交互方式，仅降低尺寸和视觉强调程度。
- 移除网盘媒体库海报墙上方的目录导航条。
- 保留本地媒体库路径下拉。
- 保留夸克、百度和 OpenList 来源配置中的统一目录选择页面，以及多根目录选择能力。

## 界面设计

### 海报卡操作按钮

- 点击区域调整为 `32 × 32`。
- 三点图标调整为 `16`。
- 圆形底色继续使用主题表面色，但降低不透明度，使按钮不再成为海报上的主要视觉元素。
- tooltip、菜单选项、点击行为和菜单定位保持不变。

### 网盘媒体库内容区

- 删除包含提供商名称、目录状态、返回上级和目录地址框的整条导航区域。
- 工具栏中的网盘来源切换、添加、管理、刷新和更多操作保持不变。
- 搜索、海报墙、扫描提示、TMDB 刮削和播放器入口保持不变。
- 网盘目录范围模型暂不删除，避免扩大本次修改范围；页面不再提供进入目录范围的入口，因此默认继续展示所选来源的全部已索引媒体。

## 数据与安全边界

本次只调整表现层，不修改网盘来源配置、已扫描索引、TMDB 元数据、字幕、播放记录或用户文件。不会修改或删除本地及网盘中的原始视频。

## 测试与验收

- 组件测试确认海报卡操作按钮使用紧凑约束和较小图标。
- 页面测试确认网盘媒体库不再渲染目录地址栏和返回上级按钮。
- 页面测试确认网盘来源工具栏、搜索和海报墙仍可使用。
- 目录选择器测试继续通过，证明夸克、百度和 OpenList 的来源配置目录选择不受影响。
- 完成全量 Flutter 测试、静态分析、Windows Release 构建和签名 MSIX 验证。
