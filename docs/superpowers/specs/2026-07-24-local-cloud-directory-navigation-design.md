# 本地与网盘目录导航设计

日期：2026-07-24

## 背景

看影音当前本地媒体库的紧凑路径框支持输入地址和按 Enter 跳转，但不能像 Windows 资源管理器一样展开当前目录的直接子文件夹。网盘媒体库则默认汇总来源下的全部媒体根目录，没有可见的目录范围导航。夸克、百度和 OpenList 虽然已有目录选择器，但布局和行为不统一，也没有与本地文件夹选择页一致的地址栏结构。

本次改动统一本地和网盘的目录导航体验，同时保留现有媒体索引、TMDB、播放与多根目录能力。

## 目标

- 本地媒体库路径框可下拉当前目录的直接子文件夹。
- 网盘媒体库提供相同风格的目录下拉，并可按目录范围过滤海报墙。
- 夸克、百度和 OpenList 的媒体目录选择页统一为与本地文件夹选择页相近的布局。
- 网盘媒体根目录继续支持多选。
- 网盘目录范围浏览优先使用已扫描索引，断网时缓存仍可浏览。
- 不改变播放器的真实路径、远程文件 ID、字幕引用、TMDB 记录或扫描结果。

## 非目标

- 不把媒体库海报墙改造成通用文件管理器。
- 不在路径下拉中显示视频、字幕或其他普通文件。
- 不移动、重命名或删除本地及网盘文件。
- 不因切换浏览目录而重新扫描或重新刮削。
- 不改变现有控件动画时长、曲线或媒体卡片交互。

## 交互设计

### 媒体库路径框

- 路径文字保持可编辑，按 Enter 继续执行现有地址跳转。
- 右侧按钮显示下拉箭头，点击后在路径框下方展示当前目录的直接子文件夹。
- 菜单项只显示文件夹名称；需要区分重名或多根目录时补充较弱的完整路径副标题。
- 选择菜单项后进入该目录，关闭菜单并更新路径文字。
- 当前目录没有子文件夹时显示不可点击的“没有子文件夹”。
- 加载期间禁用重复操作并显示进度；加载失败时保留当前路径和内容。

### 本地媒体库

- 展开下拉时按需读取当前磁盘目录，不在页面启动时递归预加载。
- 选择子文件夹后沿用现有目录进入逻辑，列表和媒体卡片行为不变。
- 空路径、无效路径和移动设备断开继续使用现有错误文案。

### 网盘媒体库

- 默认浏览范围为“全部媒体根目录”，保留当前来源级汇总视图。
- 目录下拉由已扫描索引和已配置媒体根目录构建，不额外请求网盘接口。
- 选择文件夹后，海报墙仅展示该文件夹及其子目录中的已索引视频。
- 当前范围提供上级入口；返回顶层后恢复“全部媒体根目录”。
- 搜索只作用于当前目录范围内的内容。
- TMDB 记录、海报、季度和选集继续基于原索引项关联，过滤不会复制或改写元数据。

### 网盘目录选择页

- 夸克、百度和 OpenList 使用统一页面结构：标题栏、上级按钮、地址栏、跳转按钮、子文件夹列表和顶部完成操作。
- 点击文件夹名称进入目录；右侧复选框选择或取消媒体根目录。
- 顶部完成操作显示已选数量，至少选择一个目录后可提交。
- 已选择的目录在进入其他层级后仍保持选择状态。
- OpenList 地址可直接加载；夸克和百度的手动地址从根目录逐级加载并匹配文件夹名称，以取得真实远程 ID。
- 地址解析失败、网络断开或权限不足时保留当前列表与选择状态，在地址栏下显示错误。
- 现有单选场景（例如夸克默认转存目录）继续使用单选配置，但复用相同页面布局。

## 架构设计

### 目录范围模型

新增纯 Dart 的目录范围模型，职责包括：

- 规范化 Windows 本地路径和 `/` 分隔的远程路径。
- 从配置根目录与已索引远程路径计算直接子目录。
- 判断媒体项是否属于当前目录或其子目录。
- 计算上级目录和顶层范围。

模型不访问文件系统、网盘客户端、Flutter Widget 或全局存储，便于独立测试。

### 路径下拉展示组件

路径框通过强类型回调请求子目录列表和执行选择。展示组件只负责：

- 地址编辑、Enter 提交和错误展示。
- 打开、关闭与定位下拉菜单。
- 加载、空状态和目录项渲染。

本地页面提供磁盘目录加载回调；网盘控制器提供索引目录树结果。表现层不直接依赖仓储或网盘服务。

### 网盘范围过滤

`CloudResourcesController` 增加当前浏览范围状态，并从完整索引快照派生当前范围内的索引项。作品、季度、TMDB 记录和海报墙集合基于过滤后的索引项生成；完整索引仍作为扫描、刮削和播放引用的唯一来源。

切换目录只更新派生展示状态并通知页面，不写入仓储。来源切换、根目录变更或索引重载时，如果当前范围已不存在，则安全回退到“全部媒体根目录”。

### 统一网盘目录选择器

新增共享的网盘目录选择页面和状态协调器，输入包括：

- 来源、凭据和初始目录。
- 初始选择集合及单选/多选模式。
- 强类型目录加载回调。

现有 `QuarkDirectoryPickerPage`、`BaiduDirectoryPickerPage` 和 `OpenListDirectoryPickerPage` 保留公开入口，内部适配共享页面，避免改动已有路由和调用方。OpenList 提交时继续转换为现有路径列表；夸克与百度继续返回带真实 ID 的 `CloudRemoteRef`。

## 数据流

### 本地下拉

1. 用户点击路径框右侧下拉按钮。
2. 页面按需读取当前目录的直接子文件夹。
3. 展示组件显示排序后的目录项。
4. 用户选择目录后调用现有 `_enterDirectory`。
5. 本地控制器刷新当前目录内容，路径框同步新路径。

### 网盘下拉

1. 来源加载索引快照并构建目录树。
2. 用户展开当前范围的直接子目录。
3. 用户选择目录，控制器更新当前范围。
4. 控制器从完整索引派生范围内媒体项和作品集合。
5. 海报墙重建；播放仍使用原索引中的远程 ID 和真实路径。

### 网盘目录选择

1. 编辑页把来源、临时凭据和已有选择传入共享选择器。
2. 选择器通过 `CloudLibraryController` 创建短生命周期客户端并读取直接子目录。
3. 用户导航、输入地址或切换复选框。
4. 用户完成后返回选择集合。
5. 编辑页沿用现有保存和媒体库刷新流程。

## 错误与安全边界

- 所有异步目录加载使用 generation 防止旧请求覆盖新导航状态。
- 失败不会清空当前目录内容、已有选择或完整媒体索引。
- 网盘客户端在成功或失败后均关闭。
- 日志不得记录 Cookie、令牌、请求头或完整凭据。
- 删除来源、索引或缓存仍不得删除用户原始文件。
- TMDB、网盘接口或网络不可用时，本地扫描与播放保持可用；已缓存网盘索引仍可按目录浏览。

## 测试设计

- 目录范围模型：直接子目录、路径边界、相似前缀、多根目录、上级和顶层回退。
- 本地路径栏：展开、空状态、加载错误、选择目录、Enter 地址跳转保持可用。
- 网盘控制器：默认全量、目录及子树过滤、搜索范围、范围失效回退、播放真实引用不变。
- 共享网盘选择器：进入目录、上级、地址跳转、加载错误保留内容、单选、多选和完成数量。
- 夸克、百度、OpenList：现有编辑入口、凭据传递、返回类型和根目录保存兼容。
- 全量执行 `flutter test --no-pub`、`flutter analyze --no-pub`、Windows Release 和 MSIX 清单/签名验证。
