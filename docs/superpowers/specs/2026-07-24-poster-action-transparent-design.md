# 海报卡操作按钮透明化设计

## 背景

本地和网盘海报卡右上角的三点操作按钮仍有圆形底色，在浅色海报上较为显眼，会遮挡画面并分散注意力。

## 目标

- 本地与网盘海报卡的三点操作按钮统一使用透明背景。
- 两边统一使用 `32 × 32` 点击区域和 `16` 像素三点图标。
- 保留原有菜单内容、tooltip、菜单位置和点击行为。
- 不改变页面工具栏、网盘来源管理、目录选择器或本地路径栏中的其他按钮。

## 实现边界

网盘海报卡继续由 `CloudResourcePosterWall` 提供操作菜单，本地海报卡继续由 `LocalPage` 提供操作菜单。两个入口分别移除主题表面色圆形底板，并使用相同的紧凑按钮样式；不为这两个简单入口新增共享组件。

透明背景仅指常态下不绘制圆形底色。Material 按钮原有的悬停、聚焦和点击反馈继续保留，以保证桌面鼠标操作反馈和键盘可访问性。

## 测试与验收

- 网盘海报卡测试确认按钮为 `32 × 32`、图标为 `16`，常态背景透明。
- 本地页面结构测试确认本地媒体菜单使用相同尺寸和透明背景。
- 两种菜单的原有操作入口继续存在。
- 全量 Flutter 测试、静态分析和 Windows Release 构建通过。
- 生成并验证签名 MSIX，复制到桌面。

## 数据安全

本次只修改表现层样式，不改变索引、TMDB 元数据、字幕、播放记录或来源配置，不会修改或删除本地与网盘中的原始视频。
